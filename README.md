# TalentScanAI – AI Resume Screening System

## 📌 Overview

TalentScanAI is a Machine Learning based system that helps HR teams automatically screen resumes and find the most suitable candidates for a job role.
The system analyzes resumes, extracts skills, and matches them with the job description to rank candidates based on relevance.

## 🎯 Purpose

Recruiters often receive hundreds of resumes for a single job position.
Manually reviewing all resumes takes a lot of time and effort.

TalentScanAI helps to:

* Reduce HR workload
* Speed up the hiring process
* Identify the most relevant candidates quickly

## ⚙️ Features

* Resume parsing and text extraction
* Skill extraction using NLP
* Job description matching
* Candidate ranking based on skill match
* Simple web interface for HR

## 🛠️ Tech Stack

* NLP
